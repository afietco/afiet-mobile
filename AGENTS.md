# Repository-wide agent instructions

## Engineering language

- Write all new or modified code identifiers, source filenames, technical comments, docblocks, test names, migration comments, and commit messages in English.
- Keep user-facing product copy in Turkish unless the task explicitly requests another language.
- Existing Turkish technical comments are legacy content. Do not copy or extend that style; translate a legacy comment when materially editing the code it describes.
- Before committing, review the staged diff and commit title for compliance with this rule.
